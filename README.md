# Simple Interest Calculator

A simple bash script to calculate simple interest.

## Description
This repository contains a script that calculates simple interest based on user-provided values. Simple interest is calculated by multiplying the daily interest rate by the principal by the number of days that elapse between payments.

## Formula
The formula for simple interest is:
`Simple Interest = (Principal * Rate * Time) / 100`

## Usage
To use the simple interest calculator, run the `simple-interest.sh` script and follow the prompts to enter the principal amount, interest rate, and time period.
